NY Subway Dashboard

Analysis of trip patterns, station activity, and arrival times using Tableau Desktop

Project Overview

This Tableau Desktop project visualizes New York subway operations, including trips per station, average stop positions, and arrival time distributions. The final dashboard combines all sheets for a comprehensive overview.

Sheet 1: Trips per Station

Shows the number of trips per station, highlighting the busiest stations in the network.

Sheet 2: Average Stop Position

Visualizes the average stop sequence of each station, giving insights into early vs late stops across trips.

Sheet 3: Arrival Time Analysis

Shows arrival time patterns per station, highlighting peak hours and station-level timing trends.

Full Dashboard

The complete dashboard combines all sheets, allowing interactive exploration of trip volumes, average stop positions, and arrival times.
